# ORBIT1

**Going to orbit. Open source. 2026.**

A civilian mission to launch a 1U CubeSat into low Earth orbit by December 2026. Every decision, every milestone, every failure — public.

## Live

**https://orbit1-beryl.vercel.app**

Real-time 3D globe tracking 1,200+ active satellites with live orbital propagation. Our planned sun-synchronous orbit is shown in red.

## Mission

| | |
|---|---|
| **Form factor** | 1U CubeSat (10 × 10 × 10 cm) |
| **Mass** | < 1.33 kg |
| **Orbit** | LEO ~500 km, Sun-synchronous |
| **Inclination** | 97.4° |
| **Target launch** | December 2026 |
| **Mission type** | Technology demonstration |

## Timeline

- **Day 0** (March 2026) — Public declaration. This repo.
- **Phase 1** (Mar–Apr) — Mission definition. Regulatory filing.
- **Phase 2** (Apr–May) — Satellite procurement. Ground station.
- **Phase 3** (May–Sep) — Build & test.
- **Phase 4** (Sep–Oct) — Launch integration.
- **Phase 5** (Nov–Dec) — **Launch.**

## Tech stack (this site)

- [satellite.js](https://github.com/shashwatak/satellite-js) — SGP4/SDP4 orbital propagation
- [globe.gl](https://github.com/vasturiano/globe.gl) — 3D globe visualization
- [CelesTrak](https://celestrak.org/) — Live TLE satellite catalog
- Next.js + Tailwind CSS

## Why open source?

To prove the barrier to space is lower than anyone thinks. If we can do it, so can you.
